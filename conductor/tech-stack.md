# Technology Stack

## Frontend

- **Framework:** Next.js (TypeScript)
- **Library:** React 19
- **Styling:** Tailwind CSS 4
- **UI Components:** Lucide React, Radix UI (Shadcn UI patterns)
- **Authentication & Integration:** AWS Amplify, @aws-amplify/ui-react

## Backend

- **Language:** Python 3.12+
- **Agent Framework:** Bedrock AgentCore, Strands Agents
- **Tools:** YouTube Transcript API, bedrock-agentcore-starter-toolkit
- **Package Management:** `uv`

## Infrastructure & Cloud

- **IaC:** AWS CDK (Python)
- **AI Services:** AWS Bedrock
- **Identity:** AWS Cognito
- **Runtime:** AWS Bedrock AgentCore

## Deployment

- **Options:**
  - AWS Bedrock AgentCore (via CDK)
